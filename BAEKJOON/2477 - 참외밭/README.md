## 문제 이해

이 문제는 참외밭의 면적을 계산하여, 주어진 참외밭에서 자라는 참외의 총 개수를 구하는 문제입니다. 참외밭은 특이한 모양의 육각형으로 구성되어 있으며, 각 변의 길이와 방향이 주어집니다. 이 육각형 밭은 특정한 패턴을 따르고 있어, 육각형에서 큰 직사각형과 작은 직사각형의 면적을 구한 다음, 이 둘의 차이를 이용해 전체 면적을 계산할 수 있습니다. 그 후, 1m²당 자라는 참외의 개수를 곱하여 최종적으로 참외의 총 개수를 구하게 됩니다. 문제를 해결하기 위해선, 참외밭의 경계를 따라 면적을 계산하는 방법을 이해하고 적용해야 합니다.

## 입출력 조건

- **입력 조건**:
  - 첫 번째 줄에는 1m²당 자라는 참외의 개수를 나타내는 정수 \( K \)가 주어집니다. \( K \)는 1 이상 20 이하의 값입니다.
  - 두 번째 줄부터 일곱 번째 줄까지, 참외밭의 변의 방향과 길이가 주어집니다. 각 줄은 두 개의 정수로 이루어져 있으며, 첫 번째 정수는 변의 방향(1: 동, 2: 서, 3: 남, 4: 북)을 나타내고, 두 번째 정수는 변의 길이(1 이상 500 이하)를 나타냅니다.
- **출력 조건**:
  - 계산된 참외밭의 면적에 \( K \)를 곱한 결과, 즉 총 참외의 개수를 출력합니다.

## 접근 방식

이 문제는 기하학적인 문제로, 육각형 모양의 참외밭 면적을 계산해야 합니다. 밭의 모양은 변이 6개인 특별한 육각형이지만, 사실상 두 개의 직사각형으로 구성된 형태입니다. 이 문제를 해결하기 위해 다음과 같은 접근 방식을 사용할 수 있습니다:

1. **큰 직사각형과 작은 직사각형을 구분**:
   - 주어진 변의 길이 중 가장 긴 가로와 세로 길이를 찾아 큰 직사각형의 면적으로 간주합니다.
   - 이후 작은 직사각형의 가로와 세로 길이를 찾아서 면적을 구합니다.
2. **큰 직사각형에서 작은 직사각형을 제외**:
   - 전체 참외밭 면적은 큰 직사각형의 면적에서 작은 직사각형의 면적을 뺀 값으로 구할 수 있습니다.
3. **참외의 총 개수 계산**:
   - 구한 전체 면적에 \( K \)를 곱하여 참외의 총 개수를 계산합니다.

## 풀이 과정

1. **입력 받기**:
   - 먼저 \( K \) 값을 입력받고, 참외밭의 각 변의 방향과 길이를 입력받아 리스트에 저장합니다.
2. **큰 직사각형의 가로와 세로 길이 찾기**:
   - 주어진 방향과 길이 중에서 가로와 세로를 구분하여 가장 긴 값을 각각 `max_width`와 `max_height`로 설정합니다. 이 두 값은 큰 직사각형의 가로와 세로 길이를 나타냅니다.
3. **작은 직사각형의 가로와 세로 길이 찾기**:
   - 큰 직사각형에서 제외할 작은 직사각형의 가로와 세로 길이를 구합니다. 이를 위해 각 변의 양 옆에 위치한 변들의 길이 합이 큰 직사각형의 세로 또는 가로 길이와 일치하는지 확인하여 작은 직사각형의 가로(`small_width`)와 세로(`small_height`)를 구합니다.
4. **전체 참외밭 면적 계산**:
   - 큰 직사각형의 면적에서 작은 직사각형의 면적을 뺀 값을 구하여 전체 참외밭의 면적을 계산합니다. 이 값은 `(max_width * max_height) - (small_width * small_height)`로 표현됩니다.
5. **참외의 총 개수 계산 및 출력**:
   - 계산된 참외밭의 면적에 \( K \)를 곱하여 참외의 총 개수를 계산합니다. 이 값을 최종 출력합니다.

## 코드 구현

```python
K = int(input())  # 1m²당 자라는 참외의 개수를 입력 받음
directions = []  # 변의 방향을 저장할 리스트
lengths = []  # 변의 길이를 저장할 리스트

for _ in range(6):
    direction, length = map(int, input().split())  # 방향과 길이를 입력 받음
    directions.append(direction)  # 방향을 리스트에 추가
    lengths.append(length)  # 길이를 리스트에 추가

max_width = 0  # 가장 긴 가로 길이를 저장할 변수
max_height = 0  # 가장 긴 세로 길이를 저장할 변수
for i in range(6):
    if directions[i] == 1 or directions[i] == 2:  # 동쪽(1) 또는 서쪽(2) 방향
        if lengths[i] > max_width:  # 현재 길이가 기존의 가로 최대값보다 크면
            max_width = lengths[i]  # 가로 최대값을 갱신
    elif directions[i] == 3 or directions[i] == 4:  # 남쪽(3) 또는 북쪽(4) 방향
        if lengths[i] > max_height:  # 현재 길이가 기존의 세로 최대값보다 크면
            max_height = lengths[i]  # 세로 최대값을 갱신

small_width = 0  # 작은 직사각형의 가로 길이를 저장할 변수
small_height = 0  # 작은 직사각형의 세로 길이를 저장할 변수
for i in range(6):
    if directions[i] == 1 or directions[i] == 2:  # 동쪽(1) 또는 서쪽(2) 방향
        if lengths[(i+1) % 6] + lengths[(i-1) % 6] == max_height:  # 양 옆 변의 합이 세로 최대값과 같으면
            small_width = lengths[i]  # 작은 직사각형의 가로 길이로 설정
    elif directions[i] == 3 or directions[i] == 4:  # 남쪽(3) 또는 북쪽(4) 방향
        if lengths[(i+1) % 6] + lengths[(i-1) % 6] == max_width:  # 양 옆 변의 합이 가로 최대값과 같으면
            small_height = lengths[i]  # 작은 직사각형의 세로 길이로 설정

area = (max_width * max_height) - (small_width * small_height)  # 큰 직사각형에서 작은 직사각형의 면적을 빼서 전체 면적 계산

total_melons = area * K  # 전체 면적에 1m²당 참외의 개수를 곱해 총 참외 수 계산

print(total_melons)  # 결과 출력
```
